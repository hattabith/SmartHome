# SmartHome Frontend

This directory contains the web-based dashboard for the SmartHome ecosystem.

## Overview
The frontend provides a user-friendly interface accessible from any modern web browser. It allows users to:
- Monitor real-time sensor data (temperature, humidity, power consumption).
- Control smart devices (turn on/off lights, smart plugs).
- Manage and configure automation rules.
- View historical data and charts.

## Recommended Stack
* **Framework:** Blazor WebAssembly or React (with TypeScript)
* **Real-time Updates:** SignalR

# SmartHome Documentation

This directory contains all the project-related documentation, guidelines, and architectural diagrams.

## Contents
* **Architecture Diagrams:** Visual representation of the system components and data flow.
* **API Documentation:** Details about the REST endpoints or MQTT topics used in the project.
* **Skill Files:** Configuration files like `AGENTS.md` or `RULES.md` that provide specific instructions and coding standards for AI coding assistants working on this project.

# SmartHome Mobile

This directory contains the mobile application for the SmartHome ecosystem.

## Overview
The mobile app provides a native-like experience for controlling your smart home on the go. It connects to the Backend API to deliver the same functionality as the web dashboard, optimized for touch screens and smaller displays.

## Recommended Stack
* **Framework:** .NET MAUI (Multi-platform App UI) or PWA (Progressive Web App)
* **Target Platforms:** Android, iOS
